# Pranksters

Turn-based card game prototype built in Unity using C#.

## Version
v1.0 – Console-playable prototype

## Features
- Deck generation and shuffling
- Turn management
- Favor system
- Prank completion tracking
- Multiplayer support

## Technology
- Unity
- C#
- Console interface

## Next Development Step
- Graphical tabletop interface
- Card UI
- Mouse-based interaction
