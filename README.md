# Prankster’s Revenge

Prankster’s Revenge is a turn-based card game prototype built in Unity using C#. Players compete to complete pranks by collecting specific sets of cards, offering cards as favors to gain favor points, and swapping with opponents to acquire needed cards. Prank points and favor points combine to determine the final score.

## How to Play (Prototype)

Each turn, a player may:

- Draw from the deck  
- Draw from the discard pile  
- Offer a prankster as favor  
- Swap favor with an opponent  
- Complete a prank if requirements are met  

### Goal
Complete pranks and maximize renown and favor points.
